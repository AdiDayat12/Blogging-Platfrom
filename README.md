# Personal Blogging Platform
This is a simple RESTful API with basic CRUD operations for a personal blogging platform. You can also find this project from [roadmap.sh](https://roadmap.sh/projects/blogging-platform-api)

# Tech Stack
- Java
- Spring Boot
- PostgreSQL
- Hibernate

# Features
- Create
- Read
- Update
- Delete
- Search By Tag

# Requirements
- Java 21
- Maven
- PostgreSQL

# How To Run
1. Clone the repository
2. Configure database in application.properties
3. Run the app:
   `mvn spring-boot:run`
